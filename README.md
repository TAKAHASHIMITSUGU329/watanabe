# watanabe

渡邉ひとし氏の電子書籍シリーズ執筆・企画資料フォルダです。  
第2巻「60歳からの新人生戦略」の原稿、MTG記録、表紙案、事業計画士関連資料を管理しています。

## フォルダ構成

- `01_原稿/`
  - `現行/` — 第2巻の本文原稿（第1部〜第6部 + あとがき）
  - `backup_4-7版_2026-05-13sync/` — 旧版バックアップ
- `02_設計資料/`
  - 第2巻の執筆設計書、構成案、般若心経メモ
- `03_MTG/`
  - MTGアジェンダ、議事録、録音・文字起こし、日付別のMTG資料
- `04_事業計画士/`
  - `PDF資料/` — 事業計画士関連PDF
  - `Pages資料/` — 事業計画士講座・集客導線などのPages資料
- `05_表紙デザイン/`
  - 表紙画像案、比較HTML、作業用画像
