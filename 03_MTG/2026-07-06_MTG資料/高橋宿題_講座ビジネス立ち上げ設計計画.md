# 高橋宿題2〜5 講座ビジネス立ち上げ設計・計画

- 作成日: 2026年7月6日
- 対象宿題: 高橋の宿題2〜5
- 前提: 2026年7月6日MTG議事メモ
- 目標時期: 2026年12月末〜2027年1月の講座ビジネス本格スタート

---

## 1. 今回の設計目的

今回の目的は、eラーニング講座を単体で売るのではなく、渡邉社長の北極星、電子書籍、note、ブログ、ラジオ、無料コンテンツ、コミュニティ、事業計画士講座を一本の導線にして、12月末〜1月の立ち上げにつなげることである。

結論として、最初に売るべきものは「講座」そのものではなく、渡邉社長の考え方に共感し、一緒に立ち上げたいと思う人との関係性である。

---

## 2. 宿題2: 講座ビジネス立ち上げの戦略・演出プラン

### 基本方針

講座ビジネスは、次の順番で立ち上げる。

1. 北極星を言語化する
2. 無料発信で共感と信頼をつくる
3. 無料コンテンツで体験入口をつくる
4. 5〜10人の立ち上げ仲間を集める
5. 第0期として小さく開始する
6. 実績・声・改善点をもとに第1期へ展開する

### 事業コンセプト案

経験を、AI時代の経営支援力に変える。

50代後半〜60代前半の人が、自分の経験知を棚卸しし、AI・DX・事業計画の学びを通じて、地域企業や個人事業主を支える側に回る講座ビジネスとして設計する。

### 商品の位置づけ

| 要素 | 役割 |
|---|---|
| 北極星・ミッション | 共感の中心。なぜやるのかを伝える |
| 電子書籍 | 信頼の土台。渡邉社長の思想と人生観を伝える |
| note・ブログ | 継続接点。考え方とノウハウを蓄積する |
| ラジオ | 権威性・認知拡大・人物像の発信 |
| eラーニング | 学習インフラ。AI/DXの実践教材として使う |
| 無料セミナー・動画 | 体験入口。不安を下げ、関係を深める |
| コミュニティ | 立ち上げ仲間の場。継続率と熱量を高める |
| 事業計画士講座 | 最終的な中核商品。収益化と育成の柱 |

## 3. 宿題3: 12月末〜1月スタートから逆算した全体ロードマップ

### Phase 0: 7月前半〜7月20日

目的: 設計を固め、次回MTGで合意する。

- 北極星メッセージを1本の文章にする
- ターゲット像を50代後半〜60代前半に絞って整理する
- note・ブログの初期10本タイトルを決める
- 無料コンテンツの候補を決める
- 第0期・立ち上げ仲間の呼びかけ文を作る
- ラジオの番組構成テンプレートを作る

### Phase 1: 7月下旬〜8月

目的: 発信を開始し、認知と信頼の土台をつくる。

- 週2本以上のnote・ブログ投稿を開始
- ノウハウ記事と人物像・思想記事を交互に出す
- 第1巻・第2巻の一部を無料公開し、現事業への接続コメントを添える
- AI/DX eラーニング講座の無料体験導線を整える
- ラジオ開始前の告知記事を出す
- リザーブストックの無料相談・無料セミナー導線を整える

### Phase 2: 9月

目的: 見込み客との接点を深める。

- 無料セミナーまたは無料説明会を月2回実施
- 「AIが苦手な経営者・支援者向け」の入口動画を作る
- 第0期候補者への個別声かけを始める
- ラジオ内容をnote・YouTube・ポッドキャストへ二次利用する
- メールまたはLINE登録者にステップ配信を開始する

### Phase 3: 10月

目的: 第0期の輪郭を見せ、参加意欲を高める。

- 第0期の募集予告ページを作る
- 立ち上げ仲間向け説明会を開催する
- 事業計画士講座とAI/DX eラーニングの組み合わせを提示する
- 参加候補者の課題をヒアリングし、講座内容へ反映する
- 無料コンテンツから個別相談への導線を強化する

### Phase 4: 11月

目的: 募集導線を完成させる。

- 第0期募集ページを公開
- 説明会、個別相談、申込、決済までの流れを確認する
- ラジオ・note・ブログで募集テーマを連動させる
- よくある質問、価格、参加条件、受講メリットを整理する
- 5〜10人の候補者を個別にフォローする

### Phase 5: 12月〜2027年1月

目的: 第0期を開始し、実績づくりに入る。

- 第0期を5〜10人で開始
- 初回オリエンテーションを実施
- eラーニングの使い方動画を案内する
- 受講者の声を収集する
- 第1期に向けた改善点を整理する

## 4. 宿題4: note・電子書籍コンテンツ発信の構成・順番・タイミング

### 発信の基本設計

note・ブログでは、次の4本柱を回す。

| 柱 | 内容 | 目的 |
|---|---|---|
| 北極星 | ミッション、人生観、なぜ今やるのか | 共感をつくる |
| ノウハウ | AI/DX、講座選び、事業計画、経営支援 | 役に立つ人だと伝える |
| 物語 | 渡邉社長の経験、師匠、電子書籍の背景 | 人物像を伝える |
| 導線 | 無料体験、説明会、ラジオ、立ち上げ仲間募集 | 行動につなげる |

### 電子書籍の使い方

電子書籍は全文をそのまま出すのではなく、次の形式で小出しにする。

1. 本文から一部を抜粋する
2. 今回の講座ビジネスにどうつながるかを追記する
3. 読者への問いを入れる
4. 無料体験、無料相談、ラジオ、説明会へ案内する

### 初期10本の記事案

1. なぜ私は第3巻を止めて、講座ビジネスに舵を切ったのか
2. AIが苦手な人ほど、最初に学ぶべきことは技術ではなく問いである
3. 60歳からの経験知は、AI時代の経営支援力になる
4. 北極星を持つ人だけが、講座を事業に変えられる
5. 講座が多すぎて選べない人へ。学習ルートは一緒に設計できる
6. 事業計画士とは、資格ではなく人生後半の役割である
7. ラジオを始める理由。声で届けたい人生後半の働き方
8. eラーニングは売る商品ではなく、仲間を育てる武器である
9. 第0期生ではなく、立ち上げ仲間を探しています
10. 50代後半からの不安を、次の役割に変える方法

## 5. 宿題5: ノウハウ記事と人物像・考え方の記事の交互発信設計

### 推奨リズム

最初は週2本を基本とする。

| 曜日 | 投稿タイプ | 役割 |
|---|---|---|
| 火曜 | ノウハウ記事 | 役立つ、保存される、検索される |
| 金曜 | 人物像・考え方記事 | 共感される、信頼される、応援される |

余力がある週は、日曜に短い導線記事を追加する。

### 1か月の発信例

| 週 | ノウハウ記事 | 人物像・考え方記事 |
|---|---|---|
| 第1週 | AIが苦手な人の最初の一歩 | なぜ第3巻を止めたのか |
| 第2週 | 講座の選び方と受講順序 | 北極星を持つ意味 |
| 第3週 | eラーニングを事業に組み込む方法 | 60歳からの経験知の価値 |
| 第4週 | 事業計画士が支援できること | 立ち上げ仲間を探す理由 |

### 各記事の型

#### ノウハウ記事

1. 読者の困りごとを一言で示す
2. 結論を先に伝える
3. 理由を3つに分ける
4. 具体例を出す
5. 無料体験・無料相談・関連記事へ案内する

#### 人物像・考え方記事

1. 最近の出来事または決断から入る
2. その背景にある価値観を語る
3. 読者の人生後半の悩みに接続する
4. 北極星・ミッションに戻る
5. ラジオ・講座・立ち上げ仲間募集へゆるく案内する

## 8. 最初の結論

今すぐ大きく売るのではなく、7月から9月は「共感・信頼・接点づくり」に集中する。

10月に第0期の輪郭を見せ、11月に募集導線を完成させ、12月末〜1月に5〜10人で開始する。

そのために、渡邉社長の北極星を中心に置き、ノウハウと人物像を交互に発信しながら、ラジオ、電子書籍、eラーニング、事業計画士講座を一つの物語としてつなげていく。
