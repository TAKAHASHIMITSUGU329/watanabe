## 第1章 全体導線図

### 導線の全体像

認知 → 共感 → 体験 → 関係 → 参画 → 収益 の6段階で設計する。

```
【認知】            【共感】            【体験】
ラジオ（富士産経G）   note・ブログ        無料セミナー・説明会
SNS・YouTube二次利用  電子書籍の無料公開    AI/DX eラーニング無料体験
紹介・既存人脈       北極星記事          学習ルート診断
      │                │                  │
      └────────┬───────┴──────────────────┘
               ▼
【関係】                      【参画】                【収益】
メール・LINEステップ配信  →   第0期・立ち上げ仲間   →   事業計画士®︎養成講座
無料個別相談                  コミュニティ参加          eラーニング講座販売
                                                     （リザーブストック決済）
```

### 各要素の役割と接続

| 段階 | 要素 | 役割 | 次への接続 |
|---|---|---|---|
| 認知 | ラジオ（月2回30分） | 富士産経Gのネームバリューで信頼と認知を獲得 | note・ブログへ誘導 |
| 認知 | ラジオ二次利用（YouTube・ポッドキャスト・note書き起こし） | 接触チャネルを増やす | 各SNSからnoteへ |
| 共感 | note・ブログ（週2本） | ノウハウと人物像を交互に発信 | 無料体験・説明会へ |
| 共感 | 電子書籍の章別無料公開 | 思想と人生観を伝える信頼の土台 | noteフォロー・LINE登録へ |
| 体験 | 無料セミナー・説明会（月2回） | AIへの不安を下げ、顔の見える関係をつくる | 個別相談へ |
| 体験 | eラーニング無料体験・学習ルート診断 | 「何から学ぶか」を解消する入口 | 有料講座・個別相談へ |
| 関係 | メール・LINEステップ配信 | 継続接点と育成 | 第0期募集告知へ |
| 参画 | 第0期・立ち上げ仲間コミュニティ | 5〜10人のコアメンバーの場 | 講座の共同立ち上げへ |
| 収益 | 事業計画士®︎養成講座＋eラーニング販売 | 中核商品。決済はリザーブストック（Stripe/PayPal設定済み） | 第1期拡大・実績化へ |

### 導線上の決済・インフラ（整備済み）

- リザーブストック: Stripe（コンビニ・カード・デビット・分割）、PayPal設定完了（7/5）
- 残タスク: 銀行振込の追加、無料相談・無料セミナー受付ページの整備

## 第3章 第0期募集コンセプト

### コンセプト

「受講生募集」ではなく「立ち上げ仲間募集」。教える側と教わる側ではなく、得意技を持ち寄るフラットなコミュニティとして発足する。

### 呼びかけ文（ドラフト）

> **立ち上げ仲間を探しています。**
>
> 私は、「老害ではなく、老益」「年金受給者ではなく、納税する人」という言葉を掲げて、人生後半の新しい働き方をつくる講座ビジネスを立ち上げようとしています。
>
> 50代後半から60代。会社で培った経験・スキル・人脈は、退職した途端に行き場を失います。ハローワークでは活かしきれないその経験知を、AI・DX・事業計画の学びと掛け合わせて、地域の中小企業や個人事業主を支える力に変える。それがこの講座の目的です。
>
> ただ、私は「先生と生徒」の関係で始めたいのではありません。探しているのは、第0期の受講生ではなく、一緒にこの仕組みを立ち上げる仲間です。
>
> AIが得意な人。営業が得意な人。経理が得意な人。補助金に詳しい人。何かひとつ、得意技を持ち寄ってください。足りないところは、互いに補い合えばいい。
>
> 2027年1月、5〜10人で小さく始めます。「コミュニティに入ったことで、自分の次の役割が見つかった」——そう言える場所を、一緒につくりませんか。

### 募集の運用方針

- 公募より先に、渡邉社長・川崎社長・高橋の人脈からの個別声かけを優先する（ベクトルの一致を重視）
- 声かけ → 資料送付 → 説明会 or 個別面談 → 参画確定、の4ステップ
- つながりの場はLINEオープンチャット、Facebookグループ、Slackのいずれかから選定（本日相談）

## 第4章 ラジオ番組テンプレート

### 番組基本情報

- 枠: 富士産経グループ総合研究所「ウェルビーングラボ」月2回・30分
- テーマ: 60歳以降の活躍（老害ではなく老益）
- パーソナリティ: 渡邉社長

### 30分の基本構成

| 時間 | コーナー | 内容 |
|---|---|---|
| 0:00〜3:00 | オープニング | 番組趣旨・自己紹介・今日のテーマ提示 |
| 3:00〜10:00 | 今週の北極星 | ミッション・人生観に関わる話。書籍・note記事と連動 |
| 10:00〜20:00 | メイントーク | ノウハウ or ゲスト対談（AI/DX、事業計画、経営支援、人生後半の働き方） |
| 20:00〜26:00 | リスナーへの問い | 「あなたの経験知は何ですか」等、行動を促す問いかけ |
| 26:00〜30:00 | クロージング | note・無料セミナー・立ち上げ仲間募集の案内・次回予告 |

### ゲスト構想

- コミュニティメンバー（立ち上げ仲間）を順次ゲスト出演させ、番組自体をコミュニティの発信の場にする
- 川崎社長のゲスト回も候補（営業・資産形成・補助金の得意技を紹介）

### 二次利用の流れ

1. 放送音源をポッドキャスト（Spotify等）へ転載
2. 収録映像または静止画＋音声でYouTube公開
3. 内容を書き起こし、note記事化（該当するノウハウ記事・人物像記事と相互リンク)
4. 切り抜き1〜3分をSNS（TikTok・Facebook・X）へ展開

### 残確認事項

- 契約書の締結状況と放送開始日（本日渡邉社長に確認）

## 第5章 無料コンテンツ案

### 無料コンテンツの3層構造

| 層 | コンテンツ | 目的 | 提供形態 |
|---|---|---|---|
| 入口 | AIが怖くなくなる最初の話（動画10〜15分） | AI不信・苦手意識を下げる | YouTube・LP掲載 |
| 入口 | 電子書籍の章別無料公開 | 思想・人物像で信頼をつくる | note |
| 体験 | AI/DX eラーニング無料体験 | 教材の質を体感してもらう | 講座サイト |
| 体験 | 講座サイトの歩き方（動画5〜10分） | 初回ログイン後の迷いをなくす | 講座サイト内 |
| 体験 | 学習ルート診断 | 「何から学べばよいか」を解消。ヒアリング→カリキュラム提案フローの入口 | フォーム＋個別回答 |
| 関係 | 無料セミナー・説明会（月2回） | 顔の見える関係づくり・第0期への接点 | Zoom |
| 関係 | 無料個別相談 | 課題ヒアリング・講座内容への反映 | リザーブストック予約 |

### 制作優先順位（提案）

1. **学習ルート診断**（フォームのみで開始可能。電通Gのカリキュラム提案フローをそのまま活かせる）
2. **講座サイトの歩き方動画**（受講者オンボーディングに即効性）
3. **AIが怖くなくなる最初の話**（セミナー第1回の内容を収録して転用）
4. 電子書籍の章別無料公開（渡邉社長の章選定後、随時）
