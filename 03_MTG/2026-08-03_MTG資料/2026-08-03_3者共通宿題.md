- **対象**: 渡邉ひとし社長、川崎社長、高橋

- **提出目安**: 2026年8月2日（日）までにLINEグループで共有
- **目的**: 「会社軸から自分軸へのリセット」を核に、第0期の対象者・提供価値・実施方法を具体化する

---

## 結論

次回MTGまでの宿題は、自由なアイデアの持ち寄りだけで終わらせず、**第0期を立ち上げるための判断材料を3人で揃えること**とする。

8月3日のMTGでは、次の4点を決められる状態を目指す。

1. 誰に参加してほしいか
2. 参加者にどのような変化を提供するか
3. 第0期をどのような条件で実施するか
4. 3人がそれぞれ何を担当するか

## 1. 渡邉社長の宿題

### 1-1. 講座コンセプトの言語化

「会社軸から自分軸へのリセット」について、300〜500字を目安に説明文を作成する。

説明文には、次の3点を含める。

- 誰の、どのような悩みを解決する講座なのか
- 参加前と参加後で、その人がどのように変わるのか
- なぜ事業計画士®︎、AI・DX、コミュニティが必要なのか

### 1-2. 第0期候補者の選定

第0期の「立ち上げ仲間」として声をかけたい候補者を5〜10名挙げる。

| 候補者 | 所属・関係性 | 声をかけたい理由 | 本人が抱えていそうな課題 |
|---|---|---|---|
|  |  |  |  |

### 1-3. 講座で必ず伝えたい内容の選定

講座に盛り込みたいテーマの中から、特に重要なものを3つに絞る。

候補例:

- MUST・WILL・CANによる自分軸の再設計
- 経験知の棚卸し
- 個人のミッション・ビジョン・バリュー
- 事業計画の考え方
- コミュニケーションと文章表現
- 老益として次世代の背中を押す役割

## 2. 川崎社長の宿題

### 2-1. 教材化できる経験の棚卸し

自身の経験から、講座の教材にできそうなテーマを3つ選ぶ。各テーマについて「受講者に何を持ち帰ってほしいか」を3行程度で整理する。

候補例:

- 会社員から独立するまでの意思決定
- 会社の肩書きを外した営業
- 医師との信頼関係のつくり方
- 不動産による資産形成・節税
- 補助金・助成金等の専門家との連携
- 人を紹介し、案件につなげる方法

| テーマ | 自身の具体的な経験 | 受講者に持ち帰ってほしいこと |
|---|---|---|
|  |  |  |

### 2-2. 第0期候補者の選定

第0期の「立ち上げ仲間」として声をかけたい候補者を3〜5名挙げる。医師ネットワークに限定せず、会社軸から自分軸へ移ろうとしている人も対象とする。

| 候補者 | 所属・関係性 | 声をかけたい理由 | 本人が抱えていそうな課題 |
|---|---|---|---|
|  |  |  |  |

### 2-3. 実践フィールド案の検討

受講者が座学だけで終わらず、実際の企業・個人の課題に触れられる、小さな案件または体験機会を1つ考える。

整理する項目:

- 支援対象
- 想定される課題
- 受講者が担当できること
- 必要な専門家・協力者
- 実施上の注意点

## 4. 3人共通の宿題

各自、次の問いへの回答をA4半ページ程度で用意する。

> この講座・コミュニティに「来てほしい人」と「来てほしくない人」は、どのような人か。

検討の観点:

| 来てほしい人 | 来てほしくない人 |
|---|---|
| 相手をリスペクトできる | 相手の意見を頭から否定する |
| 自分の経験を誰かのために役立てたい | 金儲けだけを第一の目的にする |
| 自ら学び、試し、改善できる | 評論だけで行動しない |
| 得意技を仲間と持ち寄れる | 自分の正解を一方的に押しつける |
| 立ち上げ段階の試行錯誤を楽しめる | 完成したサービスだけを求める |
