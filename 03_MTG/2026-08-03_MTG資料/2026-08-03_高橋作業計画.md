# 2026年8月3日MTGに向けた高橋作業計画

- **作成日**: 2026年7月20日
- **提出先**: 2026年8月3日（月）11:00〜 3者MTG
- **目的**: 第0期について、3人が具体的な判断を下せる資料を揃える

---

## 結論

高橋の役割は、講座の完成版を一人で作ることではない。渡邉社長の思想と川崎社長の実践経験を、3人が比較・検討・決定できる**たたき台**に変えることである。

作業は、次の順番で進める。

1. 講座コンセプトの1枚企画書
2. 第0期の条件比較と推奨案
3. 経験知棚卸しシート
4. 8月3日MTG用提出資料への統合

記事、ラジオ、無料診断、説明会の制作は、第0期の対象者と提供価値を決めた後に着手する。

## Step 1. 講座コンセプトの1枚企画書

**優先度: 最優先**

### 目的

講座の全体像をA4・1ページ相当で共有し、3人の共通認識を作る。

### 盛り込む項目

1. 講座の仮称
2. コアコンセプト
3. 想定する参加者
4. 参加者が現在抱えている問題
5. 参加前から参加後への変化
6. 事業計画士®︎・AI／DX・コミュニティの関係
7. 渡邉社長・川崎社長・高橋の役割
8. 第0期の位置づけ

### 完了条件

- 1ページを見れば「誰に、何を、なぜ提供するか」が説明できる
- 3人が修正意見を出せる具体度になっている
- 未決事項を決定事項のように書いていない

## Step 2. 第0期の条件比較と推奨案

### 目的

選択肢を並べるだけでなく、高橋としての推奨案を提示し、8月3日に条件を決定する。

### 比較する項目

| 項目 | 比較する選択肢 |
|---|---|
| 定員 | 5名／8名／10名 |
| 期間 | 3か月／6か月 |
| 開始時期 | 2026年12月／2027年1月 |
| 実施形式 | オンライン中心／対面併用 |
| 価格 | 無料モニター／有料モニター／通常価格 |
| 提供内容 | eラーニング、面談、グループ会、実践課題 |
| 成果物 | 自分軸宣言、経験知棚卸し、支援テーマ、事業計画、実践報告 |

### 現時点の仮説

- 定員: 5名
- 期間: 3か月
- 開始: 2027年1月
- 形式: Zoom中心＋必要に応じて対面実践会
- 位置づけ: 受講生ではなく立ち上げ仲間
- 価格: 通常価格より抑えた有料モニター

### 完了条件

- 各選択肢の利点と注意点が比較できる
- 推奨案と、その理由が説明できる
- 8月3日に決める項目が明示されている

## Step 3. 経験知棚卸しシート

### 目的

「会社軸から自分軸へのリセット」を、小さく体験できるワークを作る。第0期候補者との面談にも利用できる形にする。

### 設問案

1. これまで経験した業界・職種・役割
2. 人からよく相談されること
3. 誰かに伝えたい成功経験・失敗経験
4. 会社名や肩書を外しても残る強み
5. 今後、背中を押したい人・企業・地域
6. 本当はこれから何をしたいか
7. AI・DXを組み合わせて試したいこと

### 完了条件

- A4・1〜2ページ程度で回答できる
- 30〜45分で記入できる
- 回答を基に、本人の支援テーマを仮設定できる

## Step 4. 8月3日MTG用提出資料への統合

### 目的

作成した資料を、MTGで説明・判断しやすい順番に並べる。

### 当日の提示順

1. 講座コンセプトの1枚企画書
2. 3人が考えた「来てほしい人／来てほしくない人」
3. 第0期の条件比較と高橋推奨案
4. 3人の第0期候補者リスト
5. 経験知棚卸しシート
6. 決定事項と8月中の担当・期限

### 完了条件

- MTG中に決める事項が6項目以内に絞られている
- 決定事項をその場で記録できる欄がある
- 決定後の担当者と期限を設定できる
