# 2026年7月20日 高橋宿題

## 講座ビジネス立ち上げ実行案

- 作成日：2026年7月20日
- 前提：2026年7月6日 渡邉社長MTG議事メモ
- 対象：主に50代後半〜60代前半
- 目標：2026年12月末〜2027年1月に、第0期を5〜10人で開始する

---

## 1. 本日の結論

最初に売るものは、eラーニングや資格講座そのものではありません。

まず、渡邉社長の「北極星」に共感し、自分の経験知を次の役割へ変えたいと考える人との関係をつくります。そのうえで、無料コンテンツ、相談・説明会、eラーニング、事業計画士講座を一つの流れとして提示します。

7月から9月は、共感・信頼・接点づくりを優先します。10月に第0期の輪郭を公開し、11月に募集導線を完成させ、12月末〜2027年1月に5〜10人で開始します。

### 北極星メッセージ案

> 人生前半で培った経験知を、AI時代の経営支援力へ変え、地域の中小企業と次世代の背中を押す人を増やす。

### 事業コンセプト案

**経験を、AI時代の経営支援力に変える。**

50代後半〜60代前半の人が、経験・人間関係・判断力を棚卸しし、AI・DXと事業計画を学び直すことで、地域企業や個人事業主を支える側へ回るための学びと実践の場をつくります。

### 各接点の役割

| 接点 | 主な役割 | 次の行動 |
|---|---|---|
| ラジオ | 認知、権威性、人物像の発信 | note・無料コンテンツを見る |
| note・ブログ | 継続接点、思想とノウハウの蓄積 | LINE・メール登録、無料診断 |
| 電子書籍 | 渡邉社長の思想に対する信頼形成 | 関連記事、無料説明会 |
| 無料コンテンツ | 学ぶ不安を下げ、小さな成功体験をつくる | 個別相談、説明会 |
| 無料セミナー | 課題の自覚と解決の全体像を伝える | eラーニング体験、相談 |
| eラーニング | AI・DXの基礎と実践を支える教材 | 事業計画士講座 |
| コミュニティ | 仲間との関係、継続、実践支援 | 第0期参加 |
| 事業計画士講座 | 経験知を経営支援力へ変える | 地域企業への支援実践 |

### 導線設計の原則

1. すべての発信を、すぐ購入させる導線にはしない。
2. 最初のCTAは、無料診断・無料体験・説明会のいずれかに絞る。
3. eラーニングを「大量の講座」ではなく「目的別に選んでもらえる教材」として見せる。
4. 講座申込前に、渡邉社長の北極星と参加者のベクトルが一致するかを確認する。
5. 第0期は完成品の販売ではなく、一緒に実践モデルをつくる場と位置づける。

### 各記事の共通構成

6. 一つだけ次の行動を提示する

1. 第1巻・第2巻からテーマに合う一節を抜粋する

3. 読者への問いを一つ入れる
4. 無料診断、無料体験、ラジオ、説明会のいずれか一つへ案内する

## 4. 成果物3：第0期「立ち上げ仲間」募集コンセプト

### 募集名称案

**経験知を地域の力に変えるプロジェクト 第0期・立ち上げ仲間募集**

### 呼びかけ文案

私たちは今、50代後半から60代前半の方が、これまで培ってきた経験を次の役割へ変えるための学びと実践の場を立ち上げようとしています。

AIやDXを学ぶことだけが目的ではありません。あなたの経験、人間関係、判断力に新しい道具を加え、地域の中小企業や個人事業主の背中を押せる存在になることが目的です。

今回は、完成した講座を受講するだけの「受講生」を募集するのではありません。内容を一緒に試し、意見を出し、支援の事例をつくってくださる5〜10人の立ち上げ仲間を募集します。

次のような方とご一緒したいと考えています。

- 人生後半に、会社名や肩書だけではない役割を持ちたい方
- 自分の経験を、地域企業や次世代のために役立てたい方
- AIに関心はあるものの、何から始めるべきか迷っている方
- 資格取得だけでなく、実際の経営支援につなげたい方
- 完成した仕組みに乗るのではなく、仲間と一緒につくることを楽しめる方

第0期では、AI・DXのeラーニング、経験知の棚卸し、事業計画の考え方、経営者との対話、支援実践を組み合わせます。少人数だからこそ、一人ひとりの目的に合った学習ルートを一緒に考えます。

興味を持ってくださった方には、まず無料説明会または個別相談で、目指す方向が合っているかをお話しします。無理な勧誘は行いません。お互いのベクトルが一致することを、何より大切にします。

### 募集時に確定が必要な項目

| 項目 | 確認内容 |
|---|---|
| 開始日 | 2026年12月末または2027年1月の具体日 |
| 期間 | 3か月を基本とするか |
| 定員 | 5〜10人のうち最終定員 |
| 価格 | 第0期価格と通常価格の考え方 |
| 実施形式 | オンライン、対面、併用 |
| 提供内容 | eラーニング、面談、グループ会、実践課題 |
| 参加条件 | 対象年代、経験、活動意欲、PC環境等 |
| 成果物 | 個人の支援テーマ、事業計画、実践報告等 |

## 9. 最終確認

本計画の中心は、講座を先に完成させて販売することではありません。

渡邉社長の北極星を明確にし、その考え方に共感する人との接点を増やし、無料体験と対話を通じて5〜10人の立ち上げ仲間を集めます。そして、eラーニングを学習インフラとして活用しながら、経験知を地域企業の支援力へ変える第0期を一緒につくります。

この順番を守ることが、12月末〜2027年1月の立ち上げに向けた最重要方針です。
