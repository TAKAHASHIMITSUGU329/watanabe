# 第2巻 執筆設計書

第1巻の書き方を分析し、同じスタイル・品質で第2巻を執筆するための設計書

---

## 1. 第1巻の書き方分析結果

### 1-1. 全体構成パターン

第1巻は全6部構成で、各部の冒頭に共通の**5段階導入フレーム**を持つ:

```
① 問題提起（読者の課題を明示）
② 解決策の提示（本部で何を提供するか）
③ それが有効である理由（なぜこの方法が効くのか）
④ 第○部の内容（何を扱うか予告）
⑤ 読了時に得られるもの（読者のベネフィット）
```

その後、3〜4章の本文が続き、最後に「まとめ」で締める。

### 1-2. 各章の構造パターン（テンプレート）

```
1. 結論（先出し）
2. 理由（多くが「理由は3つ」形式）
3. 具体事例 or 比較表（図解化）
4. 派生テーマ or 詳細解説
5. 実践的示唆 or 次章への架橋
```

### 1-3. 文体

| 要素 | 第1巻のスタイル |
|------|---------------|
| 基調 | **ですます調**で統一（親近感） |
| 混在 | 一部「〜のです」「〜ならない」（である調寄り）で力感を出す |
| 一人称 | **「私」**で語る |
| 読者呼称 | **「あなた」**で直接語りかけ |
| 結論の出し方 | **「結論：」**と明示的に冒頭提示 |
| 理由の示し方 | **「理由は3つあります」**と数字で段階化 |
| 比較構造 | **「⑴」「⑵」**で視覚化 |
| 転換 | 「しかし」「だが」で切り替え |
| 断定 | 説得力のある断定的表現を要所に配置 |

### 1-4. 一節あたりの文字数

| 部分 | 文字数 |
|------|--------|
| 短い段落 | 50〜150字 |
| 標準段落 | 200〜400字 |
| 解説段落 | 300〜600字 |
| 1節全体 | 600〜1,200字 |
| 1章全体 | 3,000〜5,000字 |
| 1部全体 | 10,000〜15,000字（章数による） |
| 書籍全体 | 約70,000〜90,000字（6部合計） |

### 1-5. 事例の入れ方

| パターン | 使い方 |
|---------|--------|
| **個人体験** | 「私は…」「ある日…」で一人称の実体験を挿入 |
| **師匠の教え** | 「師匠は、こう教えてくれました」で引用 |
| **失敗事例** | 具体的な失敗→学びの構造で共感を得る |
| **成功事例** | 企業名・取り組み内容を具体的に記述 |
| **比較事例** | 従来型 vs 新しい方法を対比表で見せる |
| **配置ルール** | 抽象的な主張の直後に具体例を配置 |

### 1-6. 冒頭パターン

| タイプ | 例 |
|--------|-----|
| **設問型** | 「あなたの会社では、〜していませんか？」 |
| **結論先出し** | 「結論：〜です」 |
| **課題提示型** | 「多くの中小企業が〜に失望しています」 |
| **定義型** | 「〇〇とは何か」 |

### 1-7. 末尾パターン

| タイプ | 例 |
|--------|-----|
| **行動提示** | 「あなたが今日からできること」で3〜4項目 |
| **まとめ列挙** | 要点を4項目で整理 |
| **読者への呼びかけ** | 「〜ことを願っています」 |
| **次章への架橋** | 「この時期に〜が、次の〇〇へつながります」 |
| **チェックリスト** | 実践項目をリスト化 |

### 1-8. 名言・キーワードの使い方

- **核キーワード**を繰り返し使用して読者に刷り込む（「恩送り」「自分軸」「あり方×やり方」）
- **師匠の言葉**を直接引用し、権威と感情の両面で訴求
- **キャッチーなフレーズ**を各部のシンボルにする（「想い出の花を摘む」など）

## 2. 第2巻の執筆設計

### 2-1. 文体ルール（第1巻踏襲）

- [ ] **ですます調**を基調とする

- [ ] 結論は**「結論：」**で明示的に先出し

- [ ] 「しかし」「だが」で転換を示す

#### 第1部：還暦は終わりではない、"リボーン"の始まりだ（12,000字）

**導入の問題提起案:**
「あなたは、60歳という節目をどのように迎えようとしていますか？ 『もう現役は終わりだ』と感じていませんか？」

| 章 | タイトル | 文字数 | 結論（先出し） | 主な事例 |
|----|---------|--------|-------------|---------|
| 第1章 | リボーンとは何か ── リセットではなく自己回帰 | 4,000字 | 結論：60歳はリセットではなくリボーン（自己回帰）です | 稲盛和夫（65歳で在家得度）、国会議員の60代一年生 |
| 第2章 | 昭和モデルの限界と日本社会の気づきの遅れ | 4,000字 | 結論：「60歳定年」という呪縛は、日本経済の停滞と同根です | 賃金＝コスト vs 投資、矢沢永吉・桑田佳祐 |
| 第3章 | 「人生は二度おいしい」── 気づいた人だけが二度目を生きる | 4,000字 | 結論：人生は延長線上に生きれば一度きり。気づけば二度やれます | 獺祭・桜井博志、岡本太郎（46歳でスキー開始） |

**まとめ・今日からできること:**
1. 自分の「60歳観」を紙に書き出す
2. 「若蔵」として学びたいことを3つ挙げる
3. 尊敬する「リボーンの先達」を1人選ぶ
4. 「二度目の人生で何をしたいか」を家族に話す
